HTML での画像読み込み最適化を、`before` / `after` の比較で確認するためのサンプルです。

詳細はこの記事を読んでください。
[「1ミリ秒でも早く届けたい! HTMLで画像読込を高速化するために取り組んでいること」 - ICS MEDIA](https://ics.media/entry/221223/)

## 構成

- `index.html`: トップページ
- `before/`: 最適化前ページ
- `after/`: 最適化後ページ
- `src/`: JS、CSS、画像アセット
- `dist/`: ビルド出力

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## 整形

```bash
npm run format
```
